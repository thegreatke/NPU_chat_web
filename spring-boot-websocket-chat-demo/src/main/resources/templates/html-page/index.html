军训小贴士

萌新看过来！

1、防晒霜 防晒霜是军训必须准备的东西。 2、合脚舒适的鞋垫。 在学校会发统一的鞋子，但是这些鞋子的底都非常薄。可自备鞋垫 3、防蚊虫叮咬的药剂。 驱蚊水也需要带着，军训的时间正好是炎热的夏末，也是蚊子猖獗之时。 4、少量治疗腹泻，中暑，夏日感冒的药品(防止校医药品不够)。 5、腰带。 6、带喉糖，军队里喊口号

生活小贴士

NPUers看过来！

零食与饮料还是戒了吧，白开水跟水果才是你的明智之选，水果补充的维生素与白开水补充的水分，可以迅速修复你军训期间受伤的皮肤。 坚持锻炼身体与自习，别问我为什么，好处的真的很多，别在最青春的年纪做一只会刷剧玩手机的肥宅。花钱做到心里有数，买东西货比三家，花钱大手大脚会让你养成很不好的习惯，比如没有自制力，做人没规划等等

学习小贴士

学霸们看过来！

大一,打牢地基。观念上将“要我学”变为“我要学”,脚踏实地学好基础课程,特别是英语和计算机。在大规划下要做小计划,坚持每天记英语单词、练习口语,并从大一开始就坚定不移地学下去。根据自己的实际情况考虑是否修读双学位或辅修第二专业,并尽早做好资料准备。大一的学习任务相对轻松,可适当参加社团活动,担当一定的职务,提高自己的组织能力和交流技巧
